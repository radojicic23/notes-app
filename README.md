# Notes Web Application
- Description: Notes Web Application in Python and Flask. 📝

# Requirements
- Python 3.x
- Flask 
- Flask-SQLAlchemy
- Flask-login

# Setup and Installation
```bash
git clone <repo-url>
```

```bash
pip install -r requirements.txt
```

```bash
python main.py
```

- Link: Go to --> `http://127.0.0.1:5000`
